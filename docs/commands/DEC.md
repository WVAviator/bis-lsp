**@DEC** — Decrement Variable

Decreases the value of one or more numeric variables.

**Format**

```
@DEC[,n] v[,v,...,v] .
```

**Parameters**

`n` - Amount by which to decrease the variable value. Specify as a literal, variable, constant, or reserved word. Default = `1`. Accepts floating point or integer values.
`v` - Name of the variable to decrement.
