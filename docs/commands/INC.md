**@INC** — Increment Variable

Increases the value of one or more numeric variables.

**Format**

```
@INC[,n] v[,v,...,v] .
```

**Parameters**

`n` - Amount by which to increase the variable value. Specify as a literal, variable, constant, or reserved word. Default = `1`. Accepts floating point or integer values.
`v` - Name of the variable to increment.
